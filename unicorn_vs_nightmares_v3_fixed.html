Unicorn vs. Nightmares

A simple, editable browser game. Jump over spooky obstacles, blast swirling Nightmares with rainbows, and see how long you can last.
Lives: 7
Score: 0
Best: 0
State: Start Screen
Controls: Left/Right arrows move, Space jumps and also lets the unicorn flap/fly upward if you keep pressing it, F shoots a rainbow, P pauses, Enter starts or restarts.
To change difficulty, edit the CONFIG object near the top of the script. You can adjust lives, jump strength, level 1 speed, obstacle spacing, and rainbow size.
